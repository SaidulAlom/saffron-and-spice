# Screenshot Checklist

Use this file as a shot list before uploading the project to GitHub, a portfolio site, or LinkedIn.

## Core screenshots

1. Homepage hero in dark mode
2. Signature menu cards with hover state
3. Mobile menu or mobile homepage section
4. Testimonials grid
5. Gallery grid
6. Contact form success state
7. Cart sidebar with items added

## Optional detail shots

- Theme toggle in light mode
- Reservation modal open
- Experiences page alternating layout
- Footer CTA and contact section
- 404 page

## Recording tips

- Capture one desktop width around 1440px.
- Capture one mobile width around 390px.
- Use realistic sample cart contents before taking the commerce screenshots.
- Prefer dark mode for the hero shot and light mode for one interior page to show range.
- If you create a short screen recording, follow the order in `docs/portfolio-demo-flow.md`.
